# JSOC Streaming Module v1

## Overview
Real-time video processing and analysis module integrating Streamlit with Clarifai AI infrastructure for surveillance and monitoring systems.

## Quick Start
docker-compose up --build

** Give teh UDP streams a few seconds to get started before attempting to access the interface

Access the interface at http://localhost:8501

Import the demo_clarifai_module_config.json file to get started. (ensure you are connected to VPN as this config is set up for staging)
