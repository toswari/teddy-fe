# PS-Field-Engineering

This repository contains various projects, scripts, and resources for field engineering at Clarifai. It includes tools for customer success, rapid prototyping, sales engineering demos, monitoring, and support.

## Structure

- **CustomerSuccessManager**: Tools for managing customer success metrics and reports.
- **OpenRouter-Monitor**: Monitoring utilities for OpenRouter.
- **RapidPrototyping**: Rapid prototyping framework and examples.
- **sales_engineering**: Sales engineering demos and prompts.
- **sales_lead-gen**: Lead generation tools.
- **scripts**: Utility scripts for various tasks.
- **support**: Support-related resources.

## Getting Started

Clone the repository and explore the individual project directories for setup instructions.
